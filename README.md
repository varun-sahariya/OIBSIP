# Oasis Infobyte - Python Programming Internship Projects

This repository contains all the projects I completed during my Python Programming internship with Oasis Infobyte. I focused on building advanced, feature-rich applications that go beyond the basic requirements, incorporating full graphical user interfaces, databases, external APIs, and modern web technologies.

---

## Projects Overview

Below is a list of the projects included in this repository. Each project folder contains its own detailed README with instructions on how to run it.

### 1. Advanced BMI Calculator
A desktop application built with PyQt that calculates a user's BMI, saves their history to a SQLite database, and visualizes their progress over time with a dynamic Matplotlib graph.

### 2. Secure Password Generator
A user-friendly desktop application that generates strong, customizable passwords. It includes features like a strength indicator and one-click copy-to-clipboard functionality.

### 3. Real-Time Weather App
A desktop weather application that fetches and displays live weather data and a 5-day forecast from the OpenWeatherMap API for any city in the world.

### 4. Full-Stack Chat Application
A complete, real-time, full-stack chat application with a Python/Flask backend and a React.js frontend. Features include public/private rooms, file sharing, a persistent database history, and an integrated AI assistant powered by the Google Gemini API.
